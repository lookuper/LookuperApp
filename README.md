Lookuper App
===========
Lookuper is application that created to help track webpage changes.

Features: 
  XPath diff, 
  generate new/old data html page, 
  inline Chrome browser.

Requirenments: 
  Win XP/7/8, 
  MS .NET 4.5 (for 4.0 not tested)
